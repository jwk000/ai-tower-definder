## 1. v3.4 形态变更总览

### 1.1 v3.0 / v3.1 / v3.2 / v3.3 → v3.4 形态对照

| 维度 | v3.3（旧） | v3.4（新） | 影响等级 |
|---|---|---|---|
| 资源轴 | 4 概念：能量 / 金币 / 火花碎片 / HP | **3 资源**：能量 / 金币 / 技能点 | 🔴 MAJOR |
| Meta 永久积累 | 火花碎片关外 meta 解锁卡 + 永久科技树节点 | **彻底废弃** —— 单 Run 内闭环，死亡无回报 | 🔴 MAJOR |
| 卡牌解锁状态 | 部分卡需碎片解锁（卡池界面） | **所有卡开局即解锁**，全卡池可抽 | 🔴 MAJOR |
| 塔升级 | 关外科技树（碎片买节点，永久持有） | **本 Run 内技能树**，技能点本 Run 分配 + Run 结束清零 | 🔴 MAJOR |
| Run 失败结算 | 金币 × 1:1 转碎片入账 | **金币随 Run 结束清零**，无 meta 收益 | 🔴 MAJOR |
| 商店 UI | 单栏 4 件混排（含碎片商品） | **左右两栏：左卡池 + 右 8 槽**，全卡牌化 | 🟠 重大 |
| 商品形态 | 文字条目 / 单一格式 | **CardType.shop_item 拓展**为第 5 类卡 | 🟠 重大 |
| 路线图顶栏 | 显示碎片余额 | 不再显示，**仅显示**水晶 HP + 金币 | 🟢 小 |

### 1.2 v3.4 三资源最终定型

| 资源 | 作用域 | 跨关持续？ | Run 结束 |
|---|---|---|---|
| **能量 E** | 关内（战斗） | 否（每关重置） | 清零 |
| **金币 G** | 关间（商店/秘境/路线图） | 是（Run 内继承） | 清零（**不**再转碎片） |
| **技能点 ✦** | 关间（卡池技能树） | 是（Run 内继承） | 清零 |

> 详见 [48-shop-redesign-v34 §2](./40-presentation/48-shop-redesign-v34.md#2-资源架构v34-三资源)

### 1.3 哲学层声明

> v3.4 是对 Roguelike 哲学的回归 ── **单 Run = 完整体验单元**。

- **没有 meta 跨局积累**：死亡 = 重来，不带任何"补偿"或"渐进解锁"
- **所有卡开局即可用**：玩家面对的是"如何在 Run 内组合 + 抉择"，而非"如何积累解锁"
- **本 Run 内乐趣**：技能点 / 卡组构筑 / 节点选择都在 Run 内完成，Run 结束全部归零
- **死亡无回报，纯 Roguelike**：失败动机来自下一次更聪明的抉择，而非"再积累一些碎片"
