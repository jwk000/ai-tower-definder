# 游戏概述

> 产品定位、目标用户、核心玩法
>
> **v3.0 重写**：根据 [25-card-roguelike-refactor](../10-gameplay/10-roguelike-loop.md) 方案，定位从「独立关卡塔防」改造为「卡牌 + Roguelike 长征塔防」融合形态。

---

## 1. 产品定位

**一句话**: "把塔防做成卡牌 Roguelike 长征" — 怪物火车 × 月圆之夜 × 塔防的融合体。每局 Run 从随机卡组开局，连闯 8 关 + 终战 Boss，关与关之间在商店与秘境之间抉择，**水晶**（玩家方核心防御实体，红色悬浮辉光晶体，无敌且对入侵敌人秒杀但每杀 1 个消耗 1 HP）HP 全程继承，水晶碎裂从第 1 关重开。

### 市场差异化

| 对比 | 传统 TD | 卡牌 Roguelike | Tower Defender (v3.0) |
|------|--------|----------------|----------------------|
| 阵容来源 | 固定可选塔池 | 卡组构筑 | **本局随机抽 12 张卡组**（每次玩都不同） |
| 关卡组织 | 独立可重玩 | 节点式长征 | **连闯 8 关 + 终战的 Roguelike 长征** |
| 关卡间策略 | 无（单关结束即结算） | 弱（仅升级） | **商店 vs 秘境二选一，影响下一关** |
| 失败代价 | 重打当前关 | 全部重来 | **从第 1 关重开，但永久卡池保留** |
| 长线驱动 | 三星 + 无尽 | 卡池构筑 | **永久卡池解锁 + 流派构筑 + 最远闯关记录** |

### 参考竞品

- **怪物火车 (Monster Train)** — 卡组构筑 + 楼层放置 + Pyre 血量持续继承（**核心参考**）
- **月圆之夜 (Night of the Full Moon)** — 关间事件房间 + 风险收益博弈（**秘境参考**）
- **Kingdom Rush** — 塔防关卡精致度标杆（**关卡设计参考**）
- **Slay the Spire** — Roguelike 进度感（**永久解锁参考**）
- **皇室战争 (Clash Royale)** — 手牌拖放交互（**部署交互参考**）

Tower Defender v3.0 定位为以上四者之间的"**卡牌驱动的塔防长征**"缺口。

## 2. 目标用户

| 用户层 | 占比 | 特征 |
|--------|------|------|
| **核心用户** | 70% | 25-35 岁玩家，喜欢 Roguelike 卡组构筑，单局 30-45 分钟，追求"流派组合 + 数值滚雪球" |
| **次要用户** | 20% | 18-24 岁学生，几何美术风格吸引，喜欢研究最优解 + 收集卡池 |
| **边缘用户** | 10% | 轻度休闲玩家，需要低门槛入门引导（前 2 关刻意降低难度） |

## 3. 核心玩法闭环

### 3.1 单局 Run 循环

```
[主菜单] → 开始新 Run
   │
   ▼
[抽卡组阶段]：系统从永久解锁卡池按稀有度权重抽 12 张作为本局卡组
   │
   ▼
[关卡 1：平原]（8-10 波）
   │  ┌──────────────────────────────────────────┐
   │  │ 每波循环（关内）：                          │
   │  │   1. 每波开始抽卡，补满手牌（默认 4 张）    │
   │  │   2. 战斗中拖卡到地图部署，消耗能量 E        │
   │  │      - 单位/建筑卡：场上持续存在             │
   │  │      - 法术卡：立即生效进弃牌堆               │
   │  │   3. 敌人沿路径走，按优先级停下攻击我方目标  │
   │  │   4. 全部敌人死亡 → 结算金币 → 下一波        │
   │  │   5. 关底 Boss 击败 → 关卡通过                │
   │  └──────────────────────────────────────────┘
   │
   ▼
[关间节点：商店 vs 秘境 二选一]
   │  - 商店：花本局金币买卡 / 升级卡 / 移除卡 / 恢复血量
   │  - 秘境：随机事件，风险收益博弈
   │
   ▼
[关卡 2：森林] → 关间节点 → ... → [关卡 8：王城]
   │
   ▼
[终战 Boss 关]（无关间节点接续，超长单波，混合所有敌人）
   │
   ├→ 击败 → Run 胜利（结算英雄分、获得火花碎片）
   │
   └→ 失败 / 任一关水晶 HP=0（水晶碎裂） → Run 失败
         │
         ▼
   [Run 结算]：发放火花碎片（按到达关卡） → 返回主菜单
   主菜单可用火花碎片永久解锁新卡 / 永久升级
```

### 3.2 游戏状态机

```
RunStart
   │
   ▼
Drafting → Battle ⇄ WaveEnd → InterLevelChoosing → (Shopping | MysticEvent) → Battle ...
                                                       │
                                                       ▼
                                                  FinalBossBattle
                                                       │
                                              ┌────────┴────────┐
                                              ▼                 ▼
                                          RunVictory       RunDefeat
                                              │                 │
                                              └────────┬────────┘
                                                       ▼
                                                   RunResult → MainMenu
```

- **Drafting**: 从永久卡池抽 12 张构成本局卡组
- **Battle**: 关内单波战斗，玩家拖卡部署消耗能量，敌人按 AI 优先级行动
- **WaveEnd**: 当前波次结束，自动结算金币，2-3 秒过渡后进入下一波
- **InterLevelChoosing**: 关卡通过后弹出二选一面板（商店 / 秘境）
- **Shopping**: 商店内消费本局金币
- **MysticEvent**: 秘境事件，情境文字 + 风险收益选项
- **FinalBossBattle**: 第 8 关后直接进入终战，无关间节点
- **RunVictory / RunDefeat**: Run 结束发放火花碎片
- **RunResult**: 显示英雄分、亮点、碎片入账，返回主菜单

### 3.3 三大创新支柱

1. **卡组随机性** — 开局抽 12 张卡决定本局流派（近战墙流 / 法术爆发流 / 生产抗压流），重玩性来自卡组组合爆炸
2. **关间节点抉择** — 每关结束的商店 vs 秘境二选一是本作 Rogue 决策核心，玩家在「稳健买卡」与「冒险换强力效果」之间博弈
3. **血量继承张力** — 水晶 HP 全程不重置，前期失血（每秒杀 1 个敌人 -1 HP）会在后期付出代价，每个决策都有长期权重

## 4. 三大核心资源（认知锚点）

| 资源 | 范围 | 来源 | 消耗用途 |
|------|------|------|---------|
| **能量 E** | 关内单波 | 每波开始恢复 +5，上限 10 | 出卡部署（消耗 1-10） |
| **金币 G** | 整局 Run | 击杀敌人 + 波次结算 | 关间商店 / 秘境消耗 |
| **火花碎片** | 跨局 meta | Run 结束按到达关卡发放 | 主菜单永久解锁卡 / 永久升级 |

**关键设计**：三种资源**作用范围、来源、消耗都不交叉**（除 Run 末未花完金币按 1:1 转碎片外），认知边界清晰。

详细规则见 [06-economy-system](../10-gameplay/11-economy.md)。

## 5. 与旧版的核心差异（v2.4 → v3.0）

| 维度 | 旧版（v2.4） | 新版（v3.0） |
|------|------------|------------|
| 关卡组织 | 5 个独立关卡，单关 5-10 分钟 | 1 局 Run 连闯 8 关 + 终战，30-45 分钟 |
| 阵容来源 | 工具栏按钮选塔，按金币预算建造 | 开局抽 12 张卡组，每波抽手牌，拖卡部署 |
| 关内经济 | 金币 + 能量 + 人口三资源，被动生产 | 关内仅用能量 E（每波恢复制），金币纯累积到关间 |
| 关间过渡 | 无 | 商店 vs 秘境二选一节点 |
| 胜负 | 单关基地血归零失败，星级评定 | 整局 Run 水晶 HP=0 失败，从第 1 关重开，无星级 |
| 长线驱动 | 关卡星级 + 无尽模式 | 永久卡池解锁 + 火花碎片 + 最远闯关记录 |
| 无尽模式 | 存在 | **删除** |
