# 06 — 经济系统

> 三种核心资源：能量（关内）、金币（本局 Run）、火花碎片（跨局 meta）
>
> **v3.0 重写**：根据 [25-card-roguelike-refactor](./25-card-roguelike-refactor.md) 方案，删除被动生产链与人口资源（改为永久升级项），按「认知边界严格分层」重新设计。
>
> **数值真理源**：本文档不持有数值，所有数值（能量消耗、金币掉落、商店价格、碎片汇率等）以 [21-MDA](./21-mda-numerical-design.md) 为准。本文档仅说明**规则与结构**。

---

## 1. 三种资源（认知边界）

```
┌─────────────────────────────────────────────────────────┐
│  能量 E   ←→  关内单波范围      → 出卡部署               │
│   ↓                                                     │
│  金币 G   ←→  整局 Run 范围      → 关间商店/秘境消费      │
│   ↓ (Run 末未花完按 1:1 转换)                            │
│  火花碎片  ←→  跨局 meta 范围    → 主菜单永久解锁         │
└─────────────────────────────────────────────────────────┘
```

| 资源 | 作用范围 | 持久性 | 来源 | 消耗用途 |
|------|---------|--------|------|---------|
| **能量 E** | 关内单波 | 每波开始重置 | 每波开始恢复 +5（不超上限） | 战斗中拖卡部署消耗 |
| **金币 G** | 整局 Run | 跨波次跨关卡累积 | 击杀敌人 + 波次结算 + 秘境事件 | 关间商店买卡 / 秘境事件 |
| **火花碎片** | 跨局 meta | 永久存档 | Run 结束按到达关卡发放 + 本局未花完金币转换 | 主菜单卡池/永久升级 |

**关键设计原则**：三种资源**作用范围、来源、消耗都不互通**（除 Run 末金币→碎片转换外）。认知边界清晰，玩家不会在"该花哪种资源"上犹豫。

## 2. 能量 E（关内出卡资源）

### 2.1 规则

| 维度 | 规则 |
|------|------|
| **上限** | 默认 10（永久升级可扩展至 12） |
| **每波开始恢复** | +5（不超过上限） |
| **战斗中恢复** | **无被动恢复**（每波回合制） |
| **跨波保留** | 上一波末剩余能量保留进入下一波，受上限约束 |
| **额外恢复** | 部分特殊卡 / 秘境事件可额外回能 |

### 2.2 能量消耗示例（具体数值见 21-MDA）

| 卡类型 | 典型能量消耗 |
|--------|------------|
| 基础建筑卡（箭塔 L1） | 3 E |
| 高级建筑卡（炮塔 L1 / 激光塔 L1） | 5-6 E |
| 基础单位卡（剑士 / 弓手） | 2-3 E |
| 高级单位卡（祭司 / 召唤师） | 4-5 E |
| 法术卡（火球术 / 减速） | 2-3 E |
| AoE 法术（箭雨） | 4 E |
| 终极召唤 / 全屏冰冻 | 8-10 E |

### 2.3 能量与塔的解耦

> **重要差异**：旧版能量是「能量塔产出 → 释放主动技能」，新版能量是「每波回复制 → 出任何卡」。
>
> 能量塔（重命名「能量水晶」）作为可选建筑卡保留在卡池中，效果是「下波开始 +3 E」或「+1 能量上限」（具体见 21-MDA），但**不是默认拥有的资源生产链**。

## 3. 金币 G（本局 Run 货币）

### 3.1 流入途径

| 来源 | 触发时机 | 数值（21-MDA） |
|------|----------|---------------|
| **击杀敌人** | 敌人死亡瞬时 | 按敌人 `bounty` 配置（普通 3-8 / 精英 15-25 / Boss 80-150） |
| **波次完成** | 全部敌人死亡 | 基础 20 G + 5 G × 关卡序号 |
| **关卡通过** | 关底 Boss 击败 | 100 G + 50 G × 关卡序号 |
| **秘境事件** | 部分事件选项 | 视事件 0-200 G |
| **生产建筑卡（如金矿）** | 部署后每秒产出 | 见 21-MDA（金矿 L1 约 3 G/s） |

### 3.2 流出途径（仅在关间节点消耗）

| 节点 | 消耗用途 | 价格范围 |
|------|---------|---------|
| **商店** | 买新卡 / 升级卡 / 移除卡 / 恢复 HP / 刷新商品 | 30-200 G |
| **秘境** | 部分事件需消耗金币换取效果 | 20-150 G |

### 3.3 关内不能花金币的设计意图

**战斗中金币只攒不花**，这是 v3.0 设计核心：
- 给玩家明确的「攒钱节奏感」（关卡内只关注出卡 + 击杀）
- 关间一次性决策让"用金币干什么"成为正式策略时刻
- 简化关内 UI（顶部 HUD 只显示能量 + HP，金币以累加方式呈现）

### 3.4 初始金币

```
本局 Run 初始金币：0 G
（每局都从零开始累积，避免"开局囤金币"破坏节奏）
```

## 4. 火花碎片（跨局 meta 资源）

### 4.1 获取规则

| 触发条件 | 碎片发放 |
|---------|---------|
| Run 失败：第 1-2 关 | 10 碎片 |
| Run 失败：第 3-5 关 | 25 碎片 |
| Run 失败：第 6-8 关 | 60 碎片 |
| Run 失败：终战失败 | 100 碎片 |
| Run 胜利：终战通关 | 200 碎片 |
| 本局金币结余 | 按 1:1 比例转碎片（Run 末未花完的金币） |
| 成就解锁 | 视成就 50-500 碎片 |

### 4.2 消耗规则

| 用途 | 触发位置 | 价格范围（21-MDA） |
|------|---------|------------------|
| **永久解锁新卡** | 主菜单卡池界面 | 50-500 碎片/张（按稀有度） |
| **永久升级卡基础等级**（L1 → L2） | 主菜单卡池界面 | 300 碎片 |
| **永久升级（大本营 HP / 手牌上限 / 能量上限）** | 主菜单升级面板 | 100-500 碎片/级 |

详见 [13-save-system §3](./13-save-system.md#3-永久解锁与火花碎片)。

### 4.3 设计意图

火花碎片让"失败也有收获"，确保 Roguelike 长征的挫败感被长线进步感平衡：
- 即使第 1 关就死，也获得 10 碎片，多次累积后能解锁新卡
- 鼓励玩家"再试一次"——下一局可能因为多一张新卡而通关

## 5. 资源流向总图

```
[关内战斗]
   │ 出卡消耗
   ▼ ────────────── 能量 E（每波回满 5，上限 10）
   │
   │ 击杀掉落 + 波次结算
   ▼
   ╔═══════════════╗
   ║   金币 G      ║ ← 整局 Run 范围累积
   ╚═══════════════╝
   │
   │ 关间消费
   ▼
[商店 / 秘境]
   │
   │ Run 结束未花完
   ▼
   ╔═══════════════╗
   ║  火花碎片     ║ ← 跨局 meta 永久存档
   ╚═══════════════╝
   │ Run 结束按关卡发放（10-200）
   │
   │ 主菜单消费
   ▼
[永久卡池 / 永久升级]
```

## 6. 经济节奏（设计意图）

| 阶段 | 玩家行为 | 设计目标 |
|------|---------|---------|
| **关 1-2（开局）** | 金币紧张，仅够商店买 1-2 件商品 | 教学期，让玩家熟悉能量节奏 |
| **关 3-5（构筑）** | 金币累计中等，关间节点选择开始有重量 | 玩家开始确立流派方向 |
| **关 6-8（成型）** | 金币充裕，可在商店进行升级或移除卡精简 | 卡组成型，秘境的高风险事件成为爆发点 |
| **终战** | 资源应该接近峰值，全力一搏 | 终战的张力最大化 |

经济节奏校验：见 [21-MDA §11 经济节奏验证表](./21-mda-numerical-design.md)（按关卡的金币累积曲线 vs 商店建议消费曲线）。

## 7. 关内不可花金币的边界规则

| 旧机制 | v3.0 处理 |
|--------|---------|
| 战斗中升级塔 | **删除**：只能在商店升级整张卡 |
| 战斗中建造时扣金币 | **改为扣能量 E** |
| 战斗中回收建筑 | **保留**：但回收返还能量 E，不返还金币 |
| 战斗中购买技能 | **删除** |

### 7.1 关内回收机制（保留并改造）

```
回收单位卡的实例：
  - 返还能量：0.5 × 该卡能量消耗 × (currentHP / maxHP)
  - 建造冷却：3 秒内不可回收
  - 战斗保护：受伤/攻击中 2 秒内不可回收
```

详细公式见 21-MDA。

## 8. 已删除机制

> 以下机制在 v3.0 中**完全删除**，不再适用：

- 旧的金币被动生产链（默认拥有金矿/能量塔）→ 改为可选卡牌
- 旧的人口资源 P → 改为永久升级项「场上单位上限」（默认 8，详见 13 文档）
- 战斗中花金币建造/升级/释放技能 → 全部改用能量
- 经济节奏的"前期紧张 / 中期回本 / 后期溢出"三段曲线 → 改用 §6 的 Run 长度经济节奏
