# 05 — 技能与Buff系统

> 技能类型、Buff叠加规则、预定义效果清单

---

## 1. 技能类型

| 类型 | 触发方式 | 示例 |
|------|----------|------|
| **主动技能** | 玩家点击释放 + CD + 能量消耗 | 英雄技能 |
| **被动技能** | 始终自动生效 | 塔L3暴击、生命偷取 |
| **光环技能** | 持续作用于范围内单位 | 范围Buff/Debuff |
| **条件技能** | 满足条件自动触发 | Boss血量<50%进入二阶段 |

## 3. Buff系统

### 3.1 Buff属性

| 属性 | 说明 |
|------|------|
| **增益** | 正面效果（加ATK/HP/速度等） |
| **减益** | 负面效果（减速/减攻/眩晕等） |
| **可影响属性** | HP、ATK、Speed、Defense(护甲)、魔抗、Range、AttackSpeed |

### 3.2 Buff规则

- 同名Buff: 刷新持续时间，层数叠加（不超过上限）
- 叠加上限: 每种Buff独立设定最大叠加层数
- 永久Buff: `duration = -1`，不被计时清除
- 来源追踪: 记录来源实体ID（用于取消来源死亡时连锁清除）
- 过期自动移除

### 3.3 预定义Buff效果

| Buff | 类型 | 效果 | 来源 |
|------|------|------|------|
| **寒冰** | 减速减益 | 每层减速20%，最多5层，持续3秒 | 冰塔攻击 |
| **冰冻** | 控制减益 | 无法移动和攻击，持续1秒 | 冰塔满层触发 |
| **嘲讽** | 控制减益 | 强制攻击盾卫，持续3秒 | 盾卫技能 |
| **战吼** | 速度/攻击增益 | +30%速度 +20%ATK，持续5秒 | Boss指挥官 |

### 3.4 Buff视觉

- 实体头顶显示Buff小图标（最多3个，16×16方块）
- 增益: 血条额外金色描边
- 减益: 血条额外紫色描边
- 冰减速: 敌人颜色向蓝色渐变（按层数）
- 冰冻: 敌人变为亮青色
- Boss二阶段: 尺寸1.3x + 颜色偏红 + 头顶皇冠

## 4. 能量消耗

释放单位主动技能消耗能量：
- 低消耗技能: 10-15能量
- 中消耗技能: 15-25能量
- 高消耗技能: 25-30能量

Boss技能不消耗玩家能量（由AI自主管理冷却）。
