# 天气系统

> 5种天气类型及其对游戏数值的影响

---

## 1. 天气类型

| 天气 | ID | 说明 |
|------|-----|------|
| **晴天** | Sunny | 默认天气，阳光普照 |
| **下雨** | Rain | 大雨滂沱，火药受潮，电流增强 |
| **下雪** | Snow | 暴雪严寒，地面结冰 |
| **下雾** | Fog | 浓雾弥漫，视野受限 |
| **夜晚** | Night | 夜幕降临，暗夜生物活跃 |

---

## 2. 天气数值影响

### 2.1 各天气对塔的影响

| 塔 / 属性 | 晴天 | 下雨 | 下雪 | 下雾 | 夜晚 |
|-----------|------|------|------|------|------|
| **箭塔** ATK | — | -10% | — | -30%攻速 | — |
| **箭塔** 射程 | — | — | — | -15% | -20% |
| **炮塔** ATK | +20% | -30% | +10% | — | — |
| **冰塔** ATK | -10% | — | +30% | — | — |
| **电塔** ATK | — | +30% | -10% | — | — |
| **激光塔** ATK | -30% | +10% | — | +30% | — |
| **激光塔** 射程 | — | — | — | — | +10% |
| **蝙蝠塔** | 休眠 | 休眠 | 休眠 | ATK+50% | ATK+50% |
| **所有塔** 射程 | — | — | — | -15% | -20% |

### 2.2 各种天气对敌人的影响

| 天气 | 敌人效果 |
|------|----------|
| 晴天 | 无 |
| 下雨 | 移速 -5%（地面泥泞） |
| 下雪 | 移速 -25%（结冰路面） |
| 下雾 | 移速 -10%（视野受限） |
| 夜晚 | 移速 +18%（夜色掩护） |

### 2.3 特殊效果

| 天气 | 特殊效果 |
|------|----------|
| 晴天 | 冰塔冰冻时间 -0.3秒 |
| 下雨 | 电塔链击 +1跳，衰减降低5% |
| 下雪 | 冰冻层数上限 +1（冰塔每击+1额外层数） |
| 下雾 | 蝙蝠塔不受射程惩罚（声波探测） |
| 夜晚 | 电塔攻速 +15% |

---

## 3. 天气切换机制

- 默认开局为晴天
- 每波结束在天气池中随机切换（可重复）
- 过渡: 1.5秒渐变淡入淡出
- 关卡可配置固定天气或天气池
- 也可在波次配置中指定每波特定天气

## 4. 视觉特效

| 天气 | 特效 |
|------|------|
| 晴天 | 无 |
| 下雨 | 顶部下落蓝色水滴粒子 |
| 下雪 | 飘落白色雪花粒子 + 地面白色覆盖 |
| 下雾 | 全屏灰色半透明叠加 + 白色雾团漂移 |
| 夜晚 | 全屏深蓝色半透明叠加 + 路径微光点 |

## 5. 关卡推荐天气

| 关卡 | 推荐天气池 | 理由 |
|------|-----------|------|
| L1 平原 | Sunny, Rain, Fog | 草原天气多变 |
| L2 沙漠 | Sunny, Fog, Night | 沙漠极少雨雪，沙暴=雾 |
| L3 冰原 | Snow, Fog, Night | 极地环境，常伴暴雪极夜 |
| L4 火山 | Sunny, Rain, Fog | 火山灰=雾，热气=雨 |
| L5 城堡 | Rain, Night, Fog | 哥特城堡终年阴暗 |
