# 游戏模式

> Run 模式规则、胜负条件、关卡内难度曲线
>
> **v3.0 重写**：根据 [25-card-roguelike-refactor](./10-roguelike-loop.md) 方案，删除「独立关卡模式」与「无尽模式」，统一为「Run 长征模式」。三星评定整体删除（Run 不评星，仅按到达关卡发放火花碎片）。
>
> **数值真理源**：本文档不持有数值，所有数值（HP/速度系数/起始资源等）以 [21-MDA](../50-data-numerical/50-mda.md) 为准。

## 1. Run 模式（唯一模式）

### 1.1 规则总览

v3.0 游戏只有**一种游戏模式**：Roguelike 长征 Run。

- 单局 Run 连闯 8 个关卡 + 终战 Boss 关，共 9 关
- 每关 8-10 波，第 9 关（终战）为单波超长 boss 战
- 每局 Run 开局重新抽取 12 张卡组，每次玩内容不同
- **水晶**（玩家方核心防御实体，前称大本营）HP 在整局 Run 内**跨关卡继承**
- 任一关卡内水晶 HP 归零 → 水晶碎裂 → Run 失败，从第 1 关重开
- 水晶机制（无敌 + 秒杀敌人 + 1HP 消耗）详见 [25 §6.2](./10-roguelike-loop.md#62-水晶机制核心防御实体)
- 完成终战 → Run 胜利

### 1.2 Run 流程

```
[主菜单] → 开始新 Run
   │
   ▼
[抽卡阶段] 从永久卡池按权重抽 12 张卡作为本局卡组
   │  - Common 60% / Rare 25% / Epic 12% / Legendary 3%
   │  - 同卡可重复
   │
   ▼
[关卡 1：平原]（8 波）
   │
   ├→ 关卡通过 → [关间节点：商店 vs 秘境] → 关卡 2
   │
   └→ 水晶 HP=0 → Run 失败
   ...
[关卡 7] → 关间节点 → [关卡 8：王城] → **无关间节点** → [关卡 9：终战]
   │
   ├→ 终战胜利 → Run 胜利
   │
   └→ 水晶 HP=0 → Run 失败
```

### 1.3 关内单波循环（核心战斗循环）

```
波次开始
   │
   ▼
1. 抽卡：补满手牌至默认 4 张（永久升级可增至 8 张）
   - persistAcrossWaves=true 的卡跨波保留，不计入新抽
   │
   ▼
2. 能量结算：能量 +5，上限 10
   │
   ▼
3. 出生敌人：按波次配置生成敌人
   │
   ▼
4. 玩家拖卡部署，消耗能量
   - 单位/建筑卡：在合法格部署，作为实体进入战场
   - 法术卡：拖到目标区域立即释放，进入弃牌堆
   │
   ▼
5. 敌人 AI：沿路径行进，遇高优先级目标停下攻击
   │
   ▼
6. 全部敌人死亡 → 自动结算金币 → 2-3 秒过渡 → 下一波
```

### 1.4 水晶 HP 继承（前称大本营血量继承）

| 维度 | 规则 |
|------|------|
| 起始 HP | Run 开始时 1000（永久升级 `baseHpBonus` 可提升） |
| 跨波保留 | 一波结束后 HP 不重置 |
| 跨关保留 | 关卡通过时 HP 不重置 |
| 关间恢复 | 商店可花金币购买血药；秘境部分事件可补血 |
| 关内消耗来源 | **唯一来源**：水晶秒杀进入攻击范围内的敌人，每杀 1 个 -1 HP（Boss 不可被秒杀，故不消耗）。敌人普攻/AOE/毒等**不会扣水晶 HP**（无敌） |
| 失败判定 | HP ≤ 0 → 水晶碎裂 → Run 失败 |
| Run 末 | 不影响 meta 货币结算 |

详细数值见 [21-MDA §18](../50-data-numerical/50-mda.md)。水晶机制规则细节见 [25 §6.2](./10-roguelike-loop.md#62-水晶机制核心防御实体)。

### 1.5 关卡内难度曲线（保留 v1.1 设计）

每关内部分为 4 个难度阶段：

| 阶段 | 波次比例 | 设计目的 | 难度系数（敌人 HP/ATK 乘数） |
|------|---------|---------|----------------------------|
| **开局阶段** | 前 20% | 引导玩家熟悉本关地形 + 试出第一套部署 | 0.8× |
| **建设阶段** | 20-50% | 玩家累积能量 + 部署主防线 | 1.0×（基准） |
| **压力阶段** | 50-80% | 引入强敌，玩家必须出高价卡 | 1.3× |
| **高潮阶段** | 80-100% | 关底 Boss + 精英波 | 1.5× + Boss 机制 |

示例（10 波关卡）：波 1-2 开局 / 波 3-5 建设 / 波 6-8 压力 / 波 9-10 高潮

### 1.6 关卡间难度递增

跨关卡的难度提升通过 `LEVEL_DIFFICULTY` 乘数体现（值见 21-MDA §6）：

```
关卡 N 的敌人最终强度 = 敌人基础数值 × LEVEL_DIFFICULTY[N] × WAVE_DIFFICULTY[wave]
```

| 关卡 | LEVEL_DIFFICULTY（HP / DMG） |
|------|-----------------------------|
| L1 平原 | 0.8 / 0.8 |
| L2 森林 | 0.9 / 0.9 |
| L3 沼泽 | 1.0 / 1.0 |
| L4 雪山 | 1.1 / 1.0 |
| L5 沙漠 | 1.2 / 1.1 |
| L6 废墟 | 1.4 / 1.2 |
| L7 熔岩 | 1.6 / 1.3 |
| L8 王城 | 1.8 / 1.5 |
| L9 终战 | 2.5 / 1.8 |

详见 [07-map-level-system §3](./13-map-level.md#3-8-关--终战清单)。

### 1.7 玩家成长与难度同步

为防止"卡组成型速度 < 敌人变强速度"：

| 关键时刻 | 玩家应有能力 | 设计校验 |
|---------|------------|---------|
| 关 1-2 通关 | 卡组流派初见雏形（2-3 张主力卡） | 关 1-2 商店金币足够购买 1 张 Rare |
| 关 3-5 通关 | 1 张升级到 L2 的核心卡 + 1 个生产链 | 关 5 总收入应支持 1 次升级 |
| 关 6-7 通关 | 至少 1 张 L3 卡 + 充分能量上限 | 关 7 总收入应支持 L3 升级或购入 Epic |
| 关 8 通关 | 卡组完全成型，能稳定应对终战 | 关 7 关间节点是终战前最后准备机会 |

经济节奏校验见 [21-MDA §11 经济节奏验证表](../50-data-numerical/50-mda.md)。

### 1.8 主菜单入口

- 主菜单 → 「开始新 Run」按钮
- 如有未完成的 Run（关间存档点）→ 显示「继续 Run」按钮，状态显示当前关卡 / 水晶 HP
- 主菜单同时提供「卡池」「永久升级」「成就」「设置」入口
- **不再提供独立关卡选择**——Run 模式必须按顺序连闯

详见 [09-ui-ux](../40-presentation/40-ui-ux.md)。

## 2. 胜负条件

### 2.1 关卡内胜负

| 条件 | 结果 |
|------|------|
| 关底 Boss 击败 + 水晶 HP > 0 | 关卡通过，进入关间节点（关 8 直接进终战，终战胜利 = Run 胜利） |
| 水晶 HP ≤ 0 | Run 失败（水晶碎裂动画 → 结算面板） |

> 注：v3.0 中**没有"限时通关"**条件。波次过渡自动进行，玩家不会因为节奏慢而失败。

### 2.2 Run 胜负

| 条件 | 结果 |
|------|------|
| 通关第 9 关（终战 Boss 击败） | **Run 胜利** |
| 任一关卡水晶 HP ≤ 0 | **Run 失败**，从第 1 关重开 |
| 玩家主动放弃 Run | 视为失败，按到达关卡发放火花碎片 |

### 2.3 Run 结算

Run 结束（无论胜负）后弹出结算画面：

| 显示项 | 说明 |
|--------|------|
| 结果 | 胜利 / 失败 |
| 最远到达关卡 | 显示进度（如 7/9） |
| 总击杀 | 累计敌人击杀数 |
| Run 时长 | 用时 |
| 流派标签 | 自动识别（如"近战墙流"/"法术爆发流"，基于卡组构成） |
| 金币结余 → 碎片 | 未花完金币 1:1 转碎片 |
| 火花碎片入账 | 按到达关卡发放（10-200） |
| 解锁的成就 | 本局新解锁的成就 |
| 按钮 | "返回主菜单" / "立即开始新 Run" |

## 3. 已删除的模式与机制

> 以下机制在 v3.0 中**完全删除**：

| 机制 | 替代方案 |
|------|---------|
| **独立关卡模式**（5 关 + 关卡选择 + 重玩） | 必须按 Run 顺序连闯，重玩通过开新 Run 实现 |
| **无尽模式** | 删除。游戏深度由卡池构筑 + Roguelike 决策 + 终战难度提供 |
| **三星评定**（★1 通关 / ★2 防守 / ★3 速通） | 删除。Run 模式不评星，按"最远到达关卡"和"火花碎片"作为玩家进度衡量 |
| **关卡卡片星数显示** | 改为显示"Run 历史最佳关卡" + "Run 通关次数" |
| **重通策略**（每次取单项最佳） | 不适用 |
| **限时通关条件** | 不再存在 |
| **波次手动跳过** | 删除，波次自动过渡 |

## 4. Run 历史记录（统计而非评级）

为代替三星，主菜单提供 Run 历史统计：

| 统计项 | 说明 |
|--------|------|
| 总 Run 次数 | 全部尝试 |
| 通关 Run 次数 | 胜利 |
| 当前连败 / 最长连胜 | 连续状态 |
| 最远到达关卡 | 历史最高 |
| 最快通关时间 | 终战 Run 中最短时长 |
| 各流派完成次数 | 按自动识别的流派统计 |
| 总火花碎片获取 | meta 货币累计 |
