## 1. 设计原则与全局规范

### 1.1 三条铁律

| # | 铁律 | 失败模式（要避免的）|
|---|---|---|
| 1 | **8 关主题、敌人机制、数值体验三者全部不重复** | "雪原版的森林"、"换皮关" |
| 2 | **每关只引入 1 个新敌人 + 1 个新机制**（KR 黄金法则） | 一关塞 6 个全新机制，玩家学不动 |
| 3 | **每关有自己的"signature flavor"**（5 大波次类型选 1 主导，60–70% 波次符合主导 flavor） | 8 关都是"杂兵 + boss"的均质化 |

### 1.2 五大波次 flavor 分类（本文档统一术语）

> 来源：综合 Kingdom Rush 波表 + BTD6 round-flavor + PvZ flag-wave 提炼。本框架是本项目的关卡设计公共语言。

| flavor | 数值特征（相对档位） | 玩家被压力测的能力 | 典型波次画面 |
|---|---|---|---|
| **A. RUSH（潮涌）** | 高数量 / 低 HP / 高速 / 单一类型 / 紧密间隔 | 覆盖 / 穿透 / AoE 持续输出 | 一片小怪从生成点同时涌出 |
| **B. SIEGE（攻坚）** | 低数量 / 极高 HP / 低速 / 带护甲 | 单体爆发 / 反甲 / 优先级管理 | 几个肉墙缓慢逼近，每个都需要专门击破 |
| **C. SWARM（饱和）** | 极高数量 / 低 HP / 多类型混合 / 多路 / 宽间隔 | 全图 DPS / 战场覆盖率 | 视野塞满，杂兵从多个角度涌来 |
| **D. ELITE-SPIKE（精英突袭）** | 1 个高 HP 目标 + 5–10 个挡线护卫 | 目标优先级 / 微操 | 精英在护卫掩护下压上，护卫帮其挡子弹 |
| **E. GAUNTLET（车轮战）** | 长波 / 多子组 / 3+ 类型 / 无停歇 | 经济管理 / 卡组灵活度 | 一波接一波不喘息，敌人组合不断变化 |

加 2 个**修饰子**（可叠加在主 flavor 上）：

| 修饰子 | 含义 |
|---|---|
| **F. SUMMONER（召唤型）** | 敌人会生敌人（递归型），强制优先击杀产卵者 |
| **G. BOSS（关底）** | 单一支配性敌人，泄漏 = 重伤水晶或终结 Run |

### 1.3 数值签名 9 字段（每关必填）

每关在本文档中给出一行**相对档位签名**（不是具体数值，具体数值在 50-mda 中分配）。**两个相邻关的 9 字段至少有 5 项不同**——这是本文档对"数值体验不重复"的可验证判定。

| 字段 | 取值档位 | 说明 |
|---|---|---|
| 1. 平均敌人 HP | 极低 / 低 / 中 / 高 / 极高 | 关卡平均水平（boss 不计入） |
| 2. 平均敌人速度 | 极慢 / 慢 / 中 / 快 / 极快 | 普通杂兵的移动节奏 |
| 3. 战场密度（敌/秒） | 0.3 / 0.5 / 1.0 / 1.5 / 2.0 | 同时屏幕上的敌人数量 |
| 4. 敌人类型熵 | 单 / 双 / 三 / 四类混合 | 单关同时活跃的敌种数 |
| 5. 护甲/抗性分布 | 物理% / 魔抗% / 飞行% / 隐形% / 钻地% | 强制玩家卡组多样性 |
| 6. 节奏档（tempo profile）| Pulse（喘息）/ Sustained（持续）/ Wall-to-wall（贴脸） | 波间间隔与波长 |
| 7. 波数 | 8 / 10 / 12 / 14 | 总波次数（含 Boss 波） |
| 8. 关卡 Twist（一句话规则修饰）| 非数值文字描述 | 该关独有的「Inscryption 式」规则修饰 |
| 9. Boss 强度 | 无 / 单形态 / 二形态 / 三形态 | Boss 阶段数 |

### 1.4 8 关 flavor 序列（强制不相邻重复）

```
关1: A (RUSH)     ──教学，建立核心循环
关2: C (SWARM)    ──虫海/钻地，与 RUSH 的"覆盖压力"区分
关3: B (SIEGE)    ──护甲战，节奏放缓，喘息空间
关4: D (ELITE)    ──护卫+精英，强制优先级管理
关5: A→C 切换    ──多类型海陆空乱斗，节奏复合
关6: E (GAUNTLET) ──车轮战，经济/卡组成型检验
关7: F (SUMMONER) ──产卵者递归，杀根才有效
关8: G (BOSS)     ──多形态终战 boss，所有 flavor 浓缩
```

**核心约束**：任何两个相邻关的主导 flavor 必须不同。本序列已满足。

### 1.5 Boss 出场节奏（综合 KR + StS）

- 关 1–3：**无 Boss 战**（关末出强化波即可），让玩家建立卡组与节奏。
- 关 4：**首个 Mini-Boss**（单形态、可用既有工具击败的"毕业考"）。
- 关 5–7：**正式 Boss**（二形态，每形态对应一种被反制的卡组流派）。
- 关 8：**终战 Boss**（三形态，依次 invalidate 三种最强流派；详见 §10）。

> 与现有 `13-map-level.md` 的「每关都有 Boss」旧策略相比，本设计**降低 Boss 频率**，把每个 Boss 都做成记忆点，避免审美疲劳（来源：BTD6 高难图普遍批评"boss 太多变成杂兵"的反馈）。

## 2. 关 1 — 🌅 边境绿野（Verdant Marches）

### 2.1 主题与视觉

- **一句话定调**：王国边境的青翠平原，远处雾中藏着一座被苔藓吞噬的废弃古墓——温暖外表下的微妙不安。
- **色调**：暖黄绿（草地）+ 浅棕（土路）+ 远处一缕紫雾（古墓彩蛋）。
- **场景视觉锚点**（详见 [43-scene-decoration](../40-presentation/43-scene-decoration.md)）：
  - 田园：摇曳的麦穗、风车、栅栏、农舍篝火、散养的羊（动态环境生物）
  - 古墓彩蛋：地图右上角远景一座倾斜的石棺、半埋的骷髅手、紫色萤火飘动
  - 天气池：`[Sunny, Rain]`（明亮、温和）

### 2.2 敌人阵容（5 杂兵 + 0 Boss）

> 关 1 沿用现有 `enemies.yaml` 中的 `grunt` / `runner` / `heavy` 三只兵作底盘（兼容老存档与教学连续性），新增 2 只主题化敌人。**关 1 无独立 Boss**，关末波改为"哥布林酋长+强化哥布林"作为告别波（Boss 阶段从关 4 才正式启动）。

| ID | 名称 | 角色定位 | 调研锚点 | 引擎实现 |
|---|---|---|---|---|
| `e_goblin_scout` | 哥布林斥候 | 基础杂兵（A 主导） | KR1 Goblin（最经典 RUSH 单元） | 现有 `grunt` 重新着色为绿皮，behavior 不变 |
| `e_wolf_rider` | 狼骑兵 | 加速冲刺单元 | KR1 Worg Rider（Silveroak 经典） | 现有 `runner`，attackMode = `single_target`，速度档"快" |
| `e_orc_brute` | 兽人战士 | 中血量近战 | KR1 Orc Brute | 现有 `heavy`，护甲档"低" |
| `e_grave_crawler` 🆕 | 墓爬者 | **战斗中破甲**（KR Shieldbearer 范式） | KR Vengeance Shieldbearer | `lifecycle.onHit`：第一次受击触发 `apply_buff(self, debuff_armor_remove, duration=permanent)`。引导玩家"主动开打而非被动等待" |
| `e_goblin_chief` 🆕 | 哥布林酋长 | 关末告别精英 | KR1 Goblin King（强化兽人） | 现有 `boss_commander` 调整 HP/技能数为"加强版兽人"，去除多形态 |

> **新机制引入**：墓爬者的「战斗中破甲」是关 1 唯一新机制，教玩家"看着兵被打才上塔"的反直觉技巧。这是调研中明确被列为"高赞但被低估"的机制之一（KR Origins/Vengeance）。

### 2.3 数值签名（9 字段）

| 字段 | 档位 |
|---|---|
| 平均敌人 HP | **极低** |
| 平均敌人速度 | 中（含 1 只"快"狼骑） |
| 战场密度 | 1.0 敌/秒 |
| 类型熵 | 三类混合（杂兵 + 狼骑 + 兽人，墓爬者仅末段出现） |
| 抗性分布 | 100% 物理 / 0% 飞行 / 0% 隐形（教学纯净） |

### 2.4 波次大纲（8 波）

| 波 | flavor | 组成 | 设计目的 |
|---|---|---|---|
| 1 | A | 哥布林斥候 ×5 | 教程：拖第一张箭塔卡 |
| 2 | A | 哥布林斥候 ×8 | 检验布点 |
| 3 | A 混 | 哥布林斥候 ×6 + 兽人 ×2 | 引入"高血单位"概念 |
| 4 | A | 狼骑兵 ×4 | **首次速度尖刺**——教远端拦截 |
| 5 | A 混 | 哥布林 ×8 + 兽人 ×3 + 狼骑 ×2 | 混合压力测试 |
| 6 | A | 哥布林 ×12（间隔 0.4s 极密） | 教 AoE 卡价值 |
| 7 | A | 墓爬者 ×3 + 哥布林 ×8 | **引入新机制**：墓爬者出现，玩家发现"挡住反而不破甲" |
| 8 | A+G' | 哥布林酋长 + 兽人 ×4 + 墓爬者 ×2 | 告别波，无独立 Boss 形态 |

> 关 1 教学心智地图：**A(单)→A(单)→A(混)→A(快)→A(混)→A(密)→A(新)→A+告别**。整关只有 RUSH 一个 flavor，避免新手认知过载（来自 PvZ 第一关白天的设计参考）。

## 3. 关 2 — 🏜️ 沙漠虫潮（Dustbreach Hive）

### 3.1 主题与视觉

- **一句话定调**：被远古虫巢侵占的沙漠商道——表层是金黄沙暴和散落的古文明骨架，地下是搏动的肉膜虫卵。

### 5.3 关卡 Twist：可双向触发的陷阱（核心新机制）

- **机关定义**：关卡 YAML 在 `obstacles` 字段新增 `type: trap_falling_rock` / `trap_poison_dart` / `trap_pit`，由场景装饰系统识别。
- **触发逻辑**（在 `RuleHandlers.ts` 新增 `trigger_trap` handler）：
  - 任意单位（友方/敌方）进入陷阱格 → 触发陷阱效果
  - 陷阱效果对**所有阵营生效**（敌人踩到落石被砸、玩家士兵踩到落石也被砸）
  - 陷阱有 CD（30-60s），触发后变灰色失效，CD 后亮起重新可用
- **玩家策略**：
  - **诱敌**：把士兵留在陷阱后方，引诱敌人通过陷阱区
  - **风险**：派士兵冲锋 = 自家士兵也会被砸
  - **借鉴**：OMD（Orcs Must Die）系列的标志性玩法，调研中明确为塔防领域"最有'我在玩当下'感觉"的机制

### 7.3 关卡 Twist：旋转路径段

- **核心机制**：所有杂兵死亡时释放孢子云 → 云内未死敌人获 `spore_swell` buff（HP+30%、速度+20%、攻击力+20%）→ 这些被 buff 的敌人死亡时 **继续释放孢子云**（无限链）。
- **玩家策略要求**：
  - **远距离击杀**：让敌人在塔射程边缘死亡，使孢子云不波及后续敌群
  - **清场塔优先**：AoE 塔可一次清完一群，避免链式 buff
  - **优先杀菌母**：源头不断 = 链反应无解
- **借鉴**：KR Origins Fungus Breeder + 调研中"高赞但被低估"的孢子机制（PvZ Funguy 概念融合 KR 分裂）。

## 9. 关 8 — 👁️ 异界裂隙终战（The Veil's Edge）

### 9.1 主题与视觉

- **一句话定调**：维度边缘的扭曲虚空——之前所有关卡的现实在此撕裂，路径会随机生成传送门，几何形状在视野中飘浮变形，最深处栖息着一个不该被注视的存在。
- **色调**：深紫黑（虚空）+ 鲜紫（裂缝）+ 异色青绿（瞳孔）+ 偶尔的金黄（神圣抵抗光晕，与紫色形成强烈对比，是玩家的最后希望符号）。
- **场景视觉锚点**：
  - 异界：扭曲的几何（不规则的多边形漂浮）、紫色裂缝粒子、漂浮的眼球图腾、镜面碎片
  - 维度交错：远景能看到此前 7 关的扭曲剪影（沙漠骨架/工厂齿轮/菌林孢子等漂浮在远处虚空中），暗示"所有维度在此塌缩"
  - **可见传送门**：路径上随机位置生成紫色漩涡 portal，敌人进入后从另一个 portal 输出（路径动态变化的极致版）
  - 天气池：固定为 `Void`（虚空天气）—— 永夜 + 紫色粒子 + 偶尔的"心跳"屏幕震动

### 9.5 单超长波结构（不同于关 1-7 的 8-14 波模式）

> 关 8 沿用 `13-map-level §3.1` 的"单波 3 阶段"结构，本文档给出具体阶段内的分组。

#### 阶段 1：物理压制段（约 12 分钟，对应 Boss 形态 1）
- 持续生成混编兵（关 2-4 精英版）+ 虚空闪烁者 ×4 + 摧塔之眼 ×2
- Boss 在阶段中后段从远端缓步入场（视觉演出 + 屏幕震动）
- 玩家可见 Boss HP 条但不能直接打 Boss（直到 Boss 移动到攻击位置才进入战斗）

#### 阶段 2：召唤狂潮段（约 8-10 分钟，对应 Boss 形态 2）
- Boss 形态切换演出后，混编兵 50% 替换为关 5-7 精英版
- 摧塔之眼 ×4 + 诅咒复仇者 ×3 出场
- Boss 持续召唤摧塔之眼标记玩家最强塔（每次 2 座，玩家可在 10s 内击杀眼解除）

#### 阶段 3：精神污染段（约 5-8 分钟，对应 Boss 形态 3）
- 全场敌人速度 ×1.5（虚空精神污染光环全场生效）
- 虚空闪烁者持续从屏幕各角落空降
- Boss 持续精神控制玩家塔（每 8s 一次，4s 期间该塔攻击友军）
- 玩家必须在精神控制窗口前快速秒杀 Boss

> **设计哲学**：关 8 不是"打杂兵到 boss 出"，而是 **玩家与 Boss 全程共处**——Boss 是关 8 的舞台中心，杂兵是 Boss 的助手。这是从 KR 终战学的，最大化记忆点。
