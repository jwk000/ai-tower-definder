# 09 — UI/UX设计

> 横屏布局 | 符号优于文字 | 一眼识别

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **符号优于文字** | 场景中不显示单位名称。用形状、动画、特效让玩家一眼识别 |
| **图标化工具栏** | 底部按钮用图形符号而非文字，紧凑且直观 |
| **按需显示信息** | 选中实体时才弹出详情面板；HUD只展示必要数字 |
| **拖拽操作** | 建造/部署都用拖拽，直观的物理感 |

│  │                                                    │  │

│  ┌─────────────────────────────────────────────────────┐│

### 区域划分

| 区域 | 内容 | 文字量 |
|------|------|--------|
| 顶部HUD | 金币数/能量数/人口数/波次/敌军存活/倒计时/倍速/暂停 | 仅数字 |
| 地图区域 | 21×9格战场 + 实体（纯视觉，无名称标签） | **零** |
| 底部工具栏 | 塔/单位/陷阱/生产按钮（图形符号） | 仅hover tooltip |
| 选中信息面板 | 屏幕底部弹出，显示选中实体详情 | 按需显示 |

## 3. 交互流程

### 3.1 核心操作

| 操作 | 步骤 | 反馈 |
|------|------|------|
| **建造塔** | 点击工具栏塔图标 → 拖出幽灵预览 → 松手放置 | 合法格绿色边框，非法格红色，建造粒子特效 |
| **部署单位** | 拖拽单位图标到地图合法位置 | 同上 |
| **选中查看** | 点击地图实体 | 底部弹出信息面板（名称/等级/属性/操作按钮） |
| **升级/回收** | 选中实体 → 信息面板操作按钮 | 升级粒子特效 / 回收消散动画 |
| **移动单位** | 拖拽地图上已有单位 | 实体跟随光标，松手落位 |
| **释放技能** | 选中单位 → 底部技能按钮 → 点目标 | 技能特效 |

### 3.2 拖拽建造视觉

- 从按钮拖出时，半透明幽灵预览跟随光标
- 合法格: 绿色边框 + 浅绿填充
- 非法格: 红色边框 + 浅红填充
- 松手: 合法格→建造动画(scale 0→1弹入) / 非法格→取消

### 3.3 选中信息面板

仅选中实体时在底部显示，含：
- 实体图形符号 + 名称
- HP条 + 数值
- 核心属性（ATK/范围/攻速）
- 操作按钮（升级/回收/技能）

未选中实体时，地图区域零文字。

## 5. HUD符号规范

| 显示项 | 符号 | 格式 |
|--------|------|------|
| 金币 | 金色圆点 ● | `● 200` |
| 能量 | 蓝色菱形 ◇ | `◇ 50` |
| 人口 | 人形符号 | `👥 4/8` |
| 波次 | 旗标 | `⚑ 3/15` |
| 敌军存活 | 骷髅 | `☠ 8/30` |
| 倒计时 | 沙漏 | `⏳ 3.2s` |
| 倍速 | 文本按钮 | `[1x]` / `[2x]` |
| 暂停 | 暂停符号 | `[⏸]` |

---

## 6. 场景内零文字原则

| 旧方案 | 新方案 |
|--------|--------|
| 圆形 + "小兵"标签 | 小红圆 + 匀速步行弹跳动画 |
| 圆形 + "箭塔"标签 | 蓝方塔形 + 箭头弹道连射 |
| 六边形 + "基地"标签 | 蓝六边形 + 护盾光环脉动 |
| 圆形 + "BOSS"标签 | 大金圆 + 皇冠 + 光环 |

玩家通过**形状+颜色+动画+弹道**四要素组合，在0.5秒内识别任意单位。
