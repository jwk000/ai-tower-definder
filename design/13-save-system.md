# 15 — 存档系统

> 存档数据结构与规则

---

## 1. 存档数据

### 1.1 进度数据

保存以下内容到 LocalStorage:

- 版本号
- 每个关卡的:
  - 是否解锁
  - 是否通关
  - 获得星数（0-3）
  - 最佳到达波次
  - 最高得分
- 无尽模式最高分
- 最后游玩的关卡
- 总游玩时间（秒）

### 1.2 默认状态

- L1 平原: 默认解锁
- L2-L5: 需通过前置关卡解锁
- 无尽模式: 始终可用

---

## 2. 存档时机

| 时机 | 操作 |
|------|------|
| 关卡胜利 | 更新星级、最佳波次、最高分，连锁解锁下一关 |
| 关卡失败 | 仅更新最佳波次（不完全覆盖已有星级） |
| 无尽模式结束 | 更新最高分（如刷新纪录） |
| 每60秒 | 保存游玩时间增量 |

---

## 3. 星级评定

| 星级 | 条件 |
|------|------|
| ★ | 通关（消灭所有波次，基地存活） |
| ★★ | 限时通关 |
| ★★★ | 无损伤（无塔被摧毁 + 基地满血） |

取历史最佳成绩（不降级）。

---

## 4. 连锁解锁

- 通过L1 → 解锁L2
- 通过L2 → 解锁L3
- 通过L3 → 解锁L4
- 通过L4 → 解锁L5

---

## 5. 重置

- 支持"重置全部进度"按钮（在关卡选择界面）
- 恢复到默认初始状态
