# 11 — 游戏模式

> 关卡模式与无尽模式规则

---

## 1. 关卡模式

### 1.1 规则

- 预设地图 + 固定波次配置（10-15波）
- 每关有独特主题、敌人配置、可用塔类型
- 通关后解锁下一关
- 总计5个关卡，分布在4-5个主题章节

### 1.2 三星评定

| 星级 | 条件 |
|------|------|
| ★ | 通关（消灭所有波次敌人，基地存活） |
| ★★ | 限时通关（在时间限制内完成） |
| ★★★ | 无损伤（无任何塔被摧毁 + 基地满血） |

### 1.3 关卡选择

- 主菜单 → 关卡选择网格（3×2排列 + 无尽模式卡片）
- 每个关卡卡片显示: 名称、主题、波数、星级、最佳成绩
- 未解锁关卡显示锁定状态和解锁条件
- 点击卡片进入关卡

---

## 2. 无尽模式

### 2.1 规则

- 从已解锁地图随机选一张作为战场
- 无限波次递增，无波次上限
- 每5波出现随机Boss
- 超过20波后，每波均为Boss波 + 大量敌人

### 2.2 难度递增

- 敌人HP每波+15%
- 敌人速度每波+3%（上限200%）
- 金币掉落每波+5%
- 护甲/魔抗每波递增
- 敌人生成间隔逐渐缩短（最低0.25秒）

### 2.3 敌人类型解锁

| 波次 | 解锁敌人 |
|------|----------|
| 1+ | 小兵 |
| 2+ | 快兵 |
| 4+ | 自爆虫 |
| 6+ | 重装兵 |
| 8+ | 法师 |

### 2.4 特殊规则触发

| 波次 | 规则 |
|------|------|
| 10 | Rush（生成间隔减半） |
| 15 | Fortified + NoBuild |
| 20 | DoubleSpeed + Fortified |
| 20+（每3波） | 随机特殊规则 |

### 2.5 评分

```
得分 = 存活波次 × 100 + 击杀数 × 10
```

最高分存入存档。

---

## 3. 胜负条件

### 3.1 胜利
- 消灭关卡所有波次的敌人
- 基地（水晶）未被摧毁

### 3.2 失败
- 基地血量归零（敌人走到终点或远程摧毁）
- 特定关卡: 限时内未消灭所有敌人

### 3.3 基地
- 初始血量: 100（L5为150）
- 位于路径终点（Base tile）
- 敌人到达终点 → 对基地造成伤害
