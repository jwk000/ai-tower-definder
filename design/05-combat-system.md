# 06 — 战斗数值系统

> 伤害公式、攻速/移速公式、难度曲线、视觉规范

---

## 1. 伤害公式

### 1.1 基础伤害

```
物理伤害 = 攻击力 × (1 - 护甲 / (护甲 + 100))
魔法伤害 = 攻击力 × (1 - 魔抗 / (魔抗 + 100))
```

- 护甲常数 = 100 → 100护甲 = 50%减伤，200护甲 = 66.7%减伤
- 魔抗常数 = 100
- 采用除法减伤公式，避免线性堆叠导致零伤或免疫

### 1.2 最终伤害

```
最终伤害 = 基础伤害 × (1 + 伤害加成%) × (1 - 伤害减免%) × 暴击倍率
```

- 暴击率 = 基础暴击率 + Buff暴击率（上限80%）
- 暴击倍率 = 1.5 + Buff暴伤加成

---

## 2. 攻速公式

```
实际攻击间隔 = 基础攻击间隔 / (1 + 攻速加成%)
攻速加成上限 = 200%（最高3倍攻速）
```

示例: 箭塔L1基础攻速1.0/s，叠加100%攻速后 → 2.0/s

---

## 3. 移速公式

```
实际移速 = 基础移速 × (1 + 移速加成%) × (1 - 减速%)
移速下限 = 基础移速 × 20%（不低于20%）
```

减速不可叠加至零，最低保留20%基础移速，防止单位完全凝固。

## 6. 弹道系统

- 攻击塔发射弹道飞行到目标（非即时伤害）
- 不同塔的弹道形状:
  - 箭塔: 蓝色箭头（箭杆+三角箭头）
  - 炮塔: 橙色圆形炮弹
  - 冰塔: 蓝色菱形冰晶
  - 电塔: 黄色三角闪电
  - 激光塔: 青色矩形光束
- 弹道速度各不相同（电塔最快600px/s，炮塔最慢300px/s）

---

## 7. 血条渲染规范

- 高度: **3px**
- 偏移: 单位中心上方 **-14px**
- 颜色梯度:
  - HP > 60%: 绿色 `#4CAF50`
  - HP 30%-60%: 黄色 `#FFC107`
  - HP < 30%: 红色 `#F44336`
- 受击闪白: 100ms白色闪烁

---

## 8. 眩晕机制

- 炮塔命中后，范围内敌人进入眩晕状态
- 眩晕持续: 1.5秒
- 眩晕期间: 敌人无法移动、无法攻击
- 眩晕视觉: 敌人金色闪烁
- Boss免疫眩晕
