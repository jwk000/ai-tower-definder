# 12 — 视觉特效

> PixiJS WebGL 渲染 | 复合几何体美术 | 以符号和动态代替文字

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **形状即身份** | 每个单位有独特的**复合几何体轮廓**和符号特征组合，一眼可辨（详细规范见 [16-美术资产设计 §3](./16-art-assets-design.md#三复合几何体系统)） |
| **动效即信息** | 不需要文字，攻击动画/待机动画/死亡动画表达单位功能 |
| **色彩即阵营** | 蓝色系=我方，红色系=敌方，黄色/青色=中立 |
| **粒子即反馈** | 命中/死亡/建造/升级用粒子特效传达，不用弹窗文字 |
| **减少文字** | 场景中不显示单位名称标签。仅在UI面板中按需显示信息 |

## 2. 渲染架构

> **v2.3 更新**：EntityLayer 包含实体主体 + 头顶 HUD（血条/名字/等级），渲染逻辑在 RenderSystem 中按实体逐行处理。

```
PixiJS Stage
├── BackgroundLayer      背景底色
├── MapLayer            地图网格（tile）+ 场景装饰
├── EntityLayer         实体图形（按Y排序）+ 头顶HUD（血条/名字/等级）
├── ProjectileLayer     弹道
├── EffectLayer        粒子特效（爆炸/天气/技能）
├── UILayer            HUD + 工具栏 + 选中信息面板
└── OverlayLayer       暂停/结算
```

## 3. 单位视觉语言

### 3.1 塔类 — 复合几何体（以结构和弹道识别）

> 详细复合几何体规范见 [16-美术资产设计 §5.1](./16-art-assets-design.md#51-塔类--6种)。

| 塔 | 复合几何体结构 | 符号特征 | 弹道特效 | 一眼识别点 |
|----|----------------|----------|----------|------------|
| **箭塔** | 基座矩形 + 细长塔身矩形 + 顶部箭头三角 + 射击口矩形 + 弓臂弧线(高等级) | 塔身纤细，顶部尖锥指向天空，等级菱形标识 | 蓝色细长箭头飞行，带尾迹粒子 | 高瘦塔形 + 蓝色箭头连射 |
| **炮塔** | 基座矩形 + 短粗塔身矩形 + 圆形炮口+炮膛 + 加固钢环 + 铆钉装饰 | 炮口有后坐力动画（攻击时向后微退），等级菱形 | 橙色圆弹抛出弧线，命中时爆炸扩散 | 矮胖塔形 + 橙色炮弹 + 爆炸 |
| **冰塔** | 基座矩形 + 六边形冰晶核心 + 内切菱形 + 4-16个小冰晶环绕 + 六边冰环 | 冰晶缓慢旋转，散发冷光粒子，等级菱形 | 蓝色菱形冰晶飞行，命中时白色冰雾扩散 | 旋转冰晶 + 蓝色菱形 + 雾 |
| **电塔** | 基座矩形 + 线圈主体矩形 + 3-8圈金色线圈环 + 1-5个放电针三角 + 电极球 | 线圈间有微小电弧跳动（待机动画），等级菱形 | 黄色闪电瞬间命中（非弹道），带分支电弧 | 线圈 + 电弧跳动 + 闪电链 |
| **激光塔** | 基座矩形 + 中央晶体柱矩形 + 聚焦透镜菱形 + 1-4个副晶体柱 + 多层辉光 | 晶体柱持续发光脉动（明暗呼吸），等级菱形 | 青色光束贯穿（细长矩形），持续0.15s | 发光晶体 + 青色贯穿光束 |
| **蝙蝠塔** | 基座矩形 + 塔柱矩形 + 2-8个蝙蝠翼 + 蝙蝠身圆 + 哥特塔尖三角 + 暗影光环 | 休眠时蝙蝠翼收起（灰色），激活时展开+紫色暗影粒子，等级菱形 | 紫色蝙蝠影飞向目标，命中后飞回（吸血） | 蝙蝠翼展开/收起 + 紫色飞行影子 |

| **弓手** | 身体矩形 + 弓弧(弧线) + 弓弦线 + 箭袋矩形 + 脚下阴影椭圆 | 移动时有拉弓蓄力姿态 | 拉弓→放箭（箭矢比其他弹道更长更细） | 弓形 + 细长箭 |

| **工程师** | 身体矩形 + 扳手交叉符号 + 2个旋转齿轮 + 安全帽矩形 + 脚下阴影椭圆 | 待机时有小齿轮粒子绕身 | 锤击动画 + 目标建筑短暂金色闪烁 | 扳手 + 修复金光 |
| **刺客** | 身体矩形 + 匕首(细长三角) + 眼部面罩矩形 + 残影矩形×2 + 脚下阴影椭圆 | 移动速度快，身后有短残影拖尾 | 瞬移（消失→在目标旁出现+红色X斩击线） | 匕首 + 瞬移残影 |

## 5. 粒子特效清单

### 5.1 战斗反馈

| 特效 | 触发 | 视觉表现 | 目的 |
|------|------|----------|------|
| **命中** | 弹道到达目标 | 弹道颜色的小圆点飞散 6-10个 | 攻击命中确认 |
| **暴击** | 暴击命中 | 加大粒子 + 轻微白色闪光 | 更高伤害反馈 |
| **爆炸** | 炮塔/自爆虫 | 红色+橙色圆点爆炸扩散 15-25个 + 快速缩放消失 | AOE伤害范围 |
| **死亡** | 单位HP归零 | 单位颜色粒子收缩→爆散 8-12个 | 击杀确认 |
| **BOSS死亡** | Boss HP归零 | 金色大爆炸 + 全场0.3s闪白 + 慢动作 | 重要事件 |

### 5.2 状态反馈

| 特效 | 触发 | 视觉表现 |
|------|------|----------|
| **眩晕** | 炮塔命中 | 目标金色tint闪烁 + 头顶黄色星形符号旋转 |
| **减速** | 冰塔命中 | 目标颜色向蓝色渐变 + 脚下蓝色冰面纹 |
| **冰冻** | 冰塔满层 | 目标变亮青色 + 外包冰晶轮廓线 + 完全静止 |
| **嘲讽** | 盾卫技能 | 被嘲讽的敌人红色描边变粗 + 指向盾卫的虚线 |
| **受击** | 任何伤害 | 100ms纯白tint（闪白） |

### 5.3 经济反馈

| 特效 | 触发 | 视觉表现 |
|------|------|----------|
| **+金币** | 击杀敌人 | 金色小圆点从尸体位置飞向HUD金币区 |
| **+能量** | 击杀精英/Boss | 蓝色小圆点飞向HUD能量区 |
| **建造完成** | 放置建筑 | 金色方块粒子从基座上升 |
| **升级完成** | 建筑升级 | 白色+金色光芒粒子上升 + 建筑短暂放大 |
| **回收** | 单位出售 | 灰色方块向下消散 + 金币粒子飞回HUD |

### 5.4 天气粒子

| 天气 | 粒子表现 |
|------|----------|
| **下雨** | 蓝色细长矩形（2×8）持续下落，400-600px/s |
| **下雪** | 白色小圆（r=2-4）缓慢飘落 + 随机横向摆动 |
| **下雾** | 大型白色半透明圆（r=40-80）缓慢漂移，alpha 0.05-0.1 |
| **夜晚** | 路径微光点（r=3，alpha 0.2-0.4）缓慢明灭 |

## 6. 动画系统

### 6.1 待机动画（让场景不僵硬）

| 单位 | 待机动画 |
|------|----------|
| 塔 | 微小上下浮动（±1px正弦波），攻击塔有微小旋转或瞄准漂移 |
| 我方单位 | 微小左右摇摆（待机呼吸感） |
| 敌人行走 | 沿路径移动时身体有微小弹跳（±2px），模拟步行 |

### 6.2 过渡动画

| 场景 | 动画 |
|------|------|
| 建造 | scale 0→1，0.2s弹性缓出 |
| 死亡 | scale 1→0 + alpha 1→0，0.3s |
| Boss二阶段 | 0.5s白/红闪烁 + scale从1→1.3弹性扩张 + 0.3s全屏暂停 |
| 天气切换 | 粒子淡出→淡入，画面色调渐变，1.5s |
| 关卡进入 | 黑色遮罩alpha 1→0，0.5s |
