# 07 — 地图与关卡系统

> 网格地图、关卡配置、波次系统、随机初始化

---

## 1. 网格地图

### 1.1 尺寸规格
- 设计分辨率: **1920×1080**（横屏16:9）
- 场景区域: **21×9格**（1344×576px），**水平垂直居中显示**（水平offsetX=288，垂直offsetY动态计算=HUD之后居中于面板之上）
- Tile大小: **64×64px**
- 背景层: **全屏 1920×1080**，覆盖整个画布（天空渐变+远景+云+天气特效）
- 底部操作面板: **1344×100px**，与地图同宽，水平居中

### 1.2 Tile类型

| 类型 | 说明 | 行为 |
|------|------|------|
| `empty` | 空地 | 可建塔（需在路径邻格） |
| `path` | 路径 | 敌人行走，不可建造 |
| `blocked` | 障碍 | 不可通行不可建造 |
| `spawn` | 出生点 | 敌人生成位置 |
| `base` | 基地 | 敌人终点 |

### 1.3 建造限制
- 仅可在**路径的8方向相邻1格内**的空地上建造
- 可建地块视觉高亮（浅绿+边框）

## 2. 关卡配置

### 2.1 关卡结构

每个关卡是一个声明式配置，包含：

| 配置项 | 说明 |
|--------|------|
| 地图数据 | 21×9格 tile 布局 + 路径坐标 + 出生点/基地 |
| 波次配置 | 固定波次数量 + 每波敌人编组 + 特殊规则 |
| 起始资源 | 金币、能量、人口上限、基地HP |
| 可用单位 | 本关允许使用的塔类型和单位类型 |
| 解锁条件 | 需要通过的前置关卡 |
| 随机池 | 天气池、中立单位池（可随机部分） |
| 主题配色 | 地形颜色方案 |
| 装饰物 | 场景障碍物（纯视觉） |

### 2.2 随机初始化机制

关卡初始化时，以下内容可从配置池中随机抽取：

| 随机项 | 池来源 | 说明 |
|--------|--------|------|
| **天气** | `weatherPool` | 从允许的天气类型中随机初始天气 |
| **禁用塔** | `banPool` | 从塔类型池中随机禁用 0-N 种塔，增加策略变数 |
| **中立单位** | `neutralPool` | 从允许的中立类型中随机放置位置和数量 |
| **装饰物** | `decorationPool` | 随机散布主题装饰（树/岩石等） |
| **路径变体** | `pathVariants` | 从多个预设路径中随机选择（增加重玩性） |

```yaml
# 关卡随机配置示例
level_01:
  fixed:
    map: plains_zigzag
    waves: l1_waves
    startingGold: 200
    
  random:
    weatherPool: [Sunny, Rain]
    weatherInit: random_from_pool      # 初始天气随机
    
    banPool: ~                         # L1不禁用任何塔
    
    neutralPool:
      - type: gold_chest
        count: [1, 3]                  # 1-3个随机
        positions: random_empty_tiles  # 随机空地块
      - type: healing_spring
        count: [0, 1]                  # 0或1个
        positions: random_empty_tiles
```

|---|------|------|------|----------|--------|------|------|

### 详细描述

#### L1 平原
- 场景: 青翠草原，土路延伸，哥布林出没
- 路径: 38格Z形，弯拐处是理想建塔点
- 禁用池: 空（不禁用）
- 天气池: [Sunny, Rain]

#### L2 沙漠
- 场景: 古代商路被黄沙半掩，沙漠强盗设伏
- 路径: 48格主路径 + 18格分叉捷径
- 禁用池: 空（不禁用）
- 天气池: [Sunny, Fog, Night]

#### L3 冰原
- 场景: 永恒冰原，双线进攻
- 路径: 双spawn汇入主路径
- 禁用池: 空（不禁用）
- 天气池: [Snow, Fog, Night]
- 特殊: 路径上永久减速15%

#### L4 火山
- 场景: 岩浆河切割大地
- 路径: 被大量blocked包围的窄路径
- 禁用池: [Ice]（可能随机禁用冰塔）
- 天气池: [Sunny, Rain, Fog]
- 特殊: 所有敌人+15护甲

#### L5 城堡
- 场景: 远古要塞内部，石柱迷宫
- 路径: 折叠迷宫，~55格
- 禁用池: [Bat]（可能禁用蝙蝠塔）
- 天气池: [Rain, Night, Fog]
- 特殊: 第10波禁止建造

## 4. 波次配置

### 4.1 波次数据

每波包含:
- 波次序号
- 敌人编组列表（类型 + 数量 + 生成间隔）
- 波次开始延迟（秒）
- Boss波标记
- 特殊规则（可选）
- 出生点索引（多spawn关卡）
- 指定天气（可选，覆盖随机）

### 4.2 特殊规则

| 规则 | 效果 |
|------|------|
| `double_speed` | 本波敌人移速 +100% |
| `fortified` | 本波敌人HP +50% |
| `rush` | 敌人生成间隔减半 |
| `no_build` | 本波期间禁止建造/升级 |

### 4.3 波次流程

1. 部署结束 → 自动倒计时（开局5s，波间3s）→ 可手动跳过
2. 波次开始 → 按编组依次生成敌人
3. 全部敌人死亡 → 波次清除 → 进入波间期
4. 最后一波清除 → 关卡胜利结算

每个关卡有主题障碍物（纯视觉，不阻塞路径）：

| 主题 | 装饰物类型 |
|------|-----------|
| 草原 | 树、灌木、花丛 |
| 沙漠 | 岩石、仙人掌、枯骨 |
