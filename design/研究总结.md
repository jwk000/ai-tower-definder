# 塔防游戏单位设计研究总结

## 研究成果概览

已成功提取并整理了来自 **5 款经典塔防游戏** 的单位设计模式，生成了完整的设计参考文档。

---

## 核心发现

### 一、敌人设计的通用分类体系

#### 按属性分类（7种基础类型）
1. **普通怪** - 血量/速度/伤害均衡
2. **快速怪** - 血量少、速度快、伤害低
3. **坦克怪** - 血量多、速度慢、伤害高
4. **飞行怪** - 无法被地面塔直接攻击
5. **治疗怪** - 能为其他敌人恢复血量
6. **召唤怪** - 能生成子单位
7. **BOSS怪** - 超高血量、多种机制

#### 按护甲体系分类（Kingdom Rush模式）
- **物理护甲**：无护甲 → 轻护甲(30-50%) → 中护甲(50-70%) → 重护甲(70%+)
- **魔法抗性**：无魔抗 → 低魔抗(30%) → 中魔抗(50%) → 高魔抗(70%+)
- **特殊属性**：迷彩、重生、强化

#### 按难度递进（Bloons TD模式）
- 使用 **RBE（Red Bloon Equivalent）** 单一数值表示敌人强度
- 每10波提升1.2-1.5倍难度
- 混合敌人时按RBE比例分配

### 二、塔设计的通用分类体系

#### 按攻击方式分类（4种基础塔）
1. **箭塔** - 远程单体、攻速快、伤害中等
2. **兵营** - 近战坦克、阻挡敌人、伤害低
3. **法师塔** - 远程魔法、穿甲、攻速慢
4. **炮塔** - 范围伤害、爆炸、攻速极慢

#### 按功能分类（5种功能类型）
- **输出型**：单体伤害、AOE伤害、持续伤害
- **控制型**：减速、冻结、眩晕、击退、传送
- **防御型**：坦克、治疗、护盾
- **经济型**：资源生成、增益光环、科技树
- **特殊型**：召唤、变形、反弹

#### 升级系统对比
| 游戏 | 升级模式 | 特点 |
|------|---------|------|
| 保卫萝卜 | 线性升级（3级） | 简洁、易上手 |
| Kingdom Rush | 分支升级（2路×5级） | 策略深度中等 |
| Bloons TD 6 | 三路径交叉（3路×5级） | 策略深度最高 |

### 三、敌人-塔的克制关系矩阵

| 敌人类型 | 最优克制塔 | 次优克制塔 |
|---------|-----------|-----------|
| 普通敌人 | 任何塔 | - |
| 快速敌人 | 减速塔+高攻速塔 | 范围塔 |
| 坦克敌人 | 穿透塔+破甲塔 | 持续伤害塔 |
| 飞行敌人 | 对空塔 | 范围塔 |
| 高护甲敌人 | 破甲塔+真伤塔 | 穿透塔 |
| 治疗敌人 | 秒杀塔+控制塔 | 高伤害塔 |
| 召唤敌人 | 范围塔+群伤塔 | 持续伤害塔 |
| BOSS敌人 | 多塔配合 | 英雄+技能 |

### 四、经济系统的三种模式

#### 保卫萝卜模式：简单线性
- 金币来源：击杀敌人、关卡奖励
- 无利息机制
- 适合：休闲玩家

#### Kingdom Rush模式：科技树+利息
- 金币来源：击杀敌人、利息(5-10%)、关卡奖励
- 有科技树系统
- 适合：中核玩家

#### Bloons TD 6模式：复杂经济体系
- 金币来源：击杀敌人、香蕉农场、利息
- 经济策略：前期最大化利息 → 中期搜卡 → 后期集中资源
- 适合：硬核玩家

### 五、难度曲线设计原则

**基础公式**：
`
敌人总强度(SET) = 防御塔总输出(DPS) × 敌人在射程内的时间
`

**难度递进**：
- 1-5波：教学阶段，难度低
- 6-15波：逐步提升，引入新敌人类型
- 16-30波：中等难度，考验策略
- 31-50波：高难度，需要优化布局
- 50+波：极限难度，考验极限操作

## Tower Defender 的设计建议

### 敌人设计方案（推荐8-10种）

**基础敌人（必须）**：
1. 普通敌人：HP=100, SPD=1.0, ATK=1
2. 快速敌人：HP=50, SPD=2.0, ATK=0.5
3. 坦克敌人：HP=300, SPD=0.5, ATK=2

**进阶敌人（推荐）**：
4. 飞行敌人：HP=80, SPD=1.5, ATK=1
5. 护甲敌人：HP=150, SPD=0.8, ATK=1, 物理护甲50%
6. 魔法敌人：HP=120, SPD=1.0, ATK=1, 魔法抗性50%

**特殊敌人（可选）**：
7. 治疗敌人：HP=100, SPD=0.8, ATK=1
8. 召唤敌人：HP=150, SPD=0.6, ATK=1
9. 迷彩敌人：HP=80, SPD=1.2, ATK=1
10. BOSS敌人：HP=1000, SPD=0.3, ATK=5

### 塔设计方案（推荐6-8种）

**基础塔（必须）**：
1. 箭塔：100金币
2. 兵营：120金币
3. 法师塔：150金币

**进阶塔（推荐）**：
4. 炮塔：180金币
5. 减速塔：140金币
6. 对空塔：160金币

**特殊塔（可选）**：
7. 持续伤害塔：170金币
8. 增益塔：200金币

### 经济系统设计

推荐模式：**保卫萝卜 + Kingdom Rush 混合**

- 初始金币：200-300
- 击杀普通敌人：10-20金币
- 击杀快速敌人：15-25金币
- 击杀坦克敌人：30-50金币
- 击杀特殊敌人：50-100金币
- 关卡奖励：100-500金币

### 难度曲线设计（30波为例）

- **1-5波**：只有敌人1-3（教学）
- **6-10波**：引入敌人4-6（难度提升）
- **11-20波**：引入敌人7-9（中等难度）
- **21-30波**：引入BOSS，混合所有敌人（高难度）

## 关键设计原则

1. **克制关系明确** - 每种敌人都有最优克制塔
2. **难度平稳递进** - 避免突然跳跃
3. **多种策略可行** - 不存在唯一解
4. **经济平衡** - 前期不缺钱，后期有压力
5. **视觉反馈清晰** - 玩家能快速识别敌人和塔的类型
