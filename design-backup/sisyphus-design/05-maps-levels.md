## 1. 网格地图系统

### 尺寸规格

- 设计分辨率: **1920×1080** (横屏 16:9)
- Tile 尺寸: **64×64 像素**
- 网格规模: **30×17 格** (30 列 × 17 行)

### Tile 类型

| 类型 | 说明 | 颜色 | 行为 |
|------|------|------|------|
| `empty` | 空地 | 绿 `#3a7d44` | 可建塔 |
| `path` | 路径 | 灰褐 `#a1887f` | 敌人行走 |
| `blocked` | 障碍 | 深灰 | 不可通行不可建造 |
| `resource` | 资源点 | 金色标记 | 特殊地点 |
| `base` | 基地 | 蓝灰 | 敌人终点 |
| `spawn` | 出生点 | 红棕 | 敌人生成位置 |

### Phase 1 地图

当前 MVP 阶段使用 30×17 网格上的 Zig-zag 路径（S 形路线），包含：
- 1 个 `spawn` 点（左上区域）
- 1 个 `base` 点（右下区域）
- 若干 `path` 格形成 S 形路线
- 其余为 `empty` 格（可建塔）

## 2. 关卡配置

### 关卡数据结构

```
关卡 {
  id              唯一标识
  name            关卡名称
  theme           主题 (森林/沙漠/冰原/火山/机械)
  map_data        地图数据 (tile布局 + 中立单位)
  waves           波次配置数组
  starting_gold   初始金币
  starting_energy 初始能量
  starting_pop    初始人口上限
  available_towers  可用塔类型
  available_units   可用移动单位
  star_conditions  三星条件
  unlock_rewards   通关解锁内容
}
```

### 主题规划

| 主题 | 章节 | 关数 | 视觉特征 |
|------|------|------|----------|
| 森林 | 1 | 5关 | 绿色调，路径为土路 |
| 沙漠 | 2 | 5关 | 黄色调，路径为沙路 |
| 冰原 | 3 | 5关 | 白色调，路径为冰路 |
| 火山 | 4 | 5关 | 红黑色调，路径为熔岩 |
| 机械 | 5 | 5关 | 灰蓝色调，路径为金属 |

目标总计 20-30 关，分 4-5 个主题章节。

### 三星条件

1. **★** — 通关（消灭所有敌人，基地存活）
2. **★★** — 限时通关（在时间限制内完成）
3. **★★★** — 无损伤（无任何塔被摧毁）

## 3. 波次配置

### 波次数据结构

```
波次 {
  wave_number     波次序号
  enemies         敌人编组 [{type, count, spawn_interval}]
  spawn_delay     波次开始延迟(秒)
  special_rules   特殊规则 (如: 全敌人加速50%)
  preview_text    预览提示文字
}
```

### Phase 1 波次配置

MVP 阶段 3 波：

| 波次 | 敌人 | 数量 | 间隔(s) | 延迟(s) |
|------|------|------|---------|---------|
| 1 | 小兵 | 5 | 1.5 | 3.0 |
| 2 | 小兵 | 8 | 1.2 | 3.0 |
| 3 | 小兵 | 12 | 1.0 | 3.0 |

### 难度曲线参考

| 波次段 | 敌人强度倍率 | 金币掉落倍率 | 特征 |
|--------|-------------|-------------|------|
| 1-3 | 0.6-0.8x | 1.0x | 教学波次 |
| 4-8 | 1.0-1.3x | 1.0x | 正常难度 |
| 9-15 | 1.4-1.8x | 1.1x | 精英大量出现 |
| 16-20 | 1.9-2.5x | 1.2x | BOSS + 精英混编 |
| 20+ | 2.5x+ | 1.3x | 无尽模式递增 |

## 4. 中立单位 (计划)

| 中立单位 | 类型 | 效果 |
|----------|------|------|
| **尖刺陷阱** | 机关 | 敌人踩到受伤害+减速 |
| **治疗泉水** | 资源点 | 周围单位/建筑缓慢回血 |
| **金币宝箱** | 资源点 | 可攻击，击破掉落大量金币 |
| **中立炮台** | 可争夺 | 谁的单位靠近就归谁控制 |
| **传送门** | 机关 | 敌人进入→从另一个出口出现 |
| **滚石** | 机关 | 定时滚下，碾压路径上的所有单位 |

## 5. 渲染规范

### 地图渲染 (Phase 1 实现)

- 空地块: `#3a7d44` (亮绿)
- 路径: `#a1887f` (灰褐)
- 基地地块: 六边形 + "基地" 文字标签
- 塔: 蓝色圆圈 + 文字标签（当前 "箭塔"）
- 敌人: 红色圆圈 + 文字标签（当前 "小兵"）
- 弹道: 蓝色箭头形状

Rendering via `RenderSystem` → command buffer → `Renderer.endFrame()` 批量绘制。
