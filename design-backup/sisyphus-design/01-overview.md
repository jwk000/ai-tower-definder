# 01 — 概述与市场定位

> 来源: GDD §§1, 3 | 版本: v0.2

---

## 1. 产品定位

> **"可以亲手操控战场的塔防游戏"** — 传统TD的框架下，加入可移动单位操控、资源链经济和实时技能释放，让每一波敌人都不无聊。

### 市场分析

| 维度 | 分析 |
|------|------|
| **品类现状** | 塔防 (TD) 是成熟品类，但近年缺乏现象级创新。主流产品分两派：Kingdom Rush 式（精致关卡，被动观战）和 Vampire Survivors 式（幸存者like，反向弹幕）。两者之间缺乏"有操作深度但不累"的中间地带 |
| **机会窗口** | 移动端横屏策略游戏供给不足。玩家对"放置类"疲劳，渴望有操作感的轻策略体验 |
| **竞争差异化** | 传统TD = 布阵 + 观战 → **我们的创新** = 布阵 + 实时操作 + 经济链 + 动态对抗 |

### 参考竞品

- Kingdom Rush 系列 — 精致关卡标杆，但玩家布阵后被动等待
- 明日方舟 — 塔防+RPG融合，但门槛偏高
- 皇室战争 — 实时对战策略，操作密度过大
- Vampire Survivors — 反向弹幕，爽快但深度不足

Tower Defender 定位为以上四者之间的"轻策略操作"缺口。

## 2. 目标用户画像

**核心用户 (70%)：**
- 25-35 岁男性，一二线城市
- 策略游戏经验（玩过 Kingdom Rush / 皇室战争 / 明日方舟）
- 单次游戏时长 15-30 分钟
- 在通勤 / 午休 / 睡前碎片时间玩
- 追求"有脑但不上头"的体验

**次要用户 (20%)：**
- 18-24 岁学生群体
- 被几何极简美术风格吸引
- 喜欢研究最优解、追求无尽模式高分

**边缘用户 (10%)：**
- 轻度休闲玩家，被简单几何画风吸引
- 需要低门槛入门引导

## 3. 核心玩法闭环

### 3.1 单局循环

```
                波次开始
                   │
    ┌──────────────┴──────────────┐
    │       部署/调整阶段          │
    │  · 建造/升级/修复塔         │
    │  · 部署/移动可移动单位      │
    │  · 使用技能/激活机关        │
    │  · 分配资源                 │
    └──────────────┬──────────────┘
                   │ 玩家点击「开始波次」
    ┌──────────────┴──────────────┐
    │       战斗阶段               │
    │  · 敌人沿路径移动            │
    │  · 塔自动攻击 (弹道飞行)     │
    │  · 玩家操控移动单位          │← 核心操作层
    │  · 实时释放技能              │
    │  · 收集击杀金币              │
    └──────────────┬──────────────┘
                   │ 所有敌人死亡 / 基地被摧毁
    ┌──────────────┴──────────────┐
    │       波次结算               │
    │  · 剩余血量奖励金币          │
    │  · 显示下一波敌人预览        │
    │  · 资源产出结算              │
    └──────────────┬──────────────┘
                   │
         ┌────────┴────────┐
         │ 不是最后一波    │──→ 回到部署阶段
         └─────────────────┘
                   │ 最后一波
    ┌──────────────┴──────────────┐
    │       关卡结算               │
    │  · 星级评定 (1-3星)         │
    │  · 解锁新内容               │
    │  · 排行榜记录               │
    └─────────────────────────────┘
```

### 3.2 游戏模式

#### 关卡模式 (Level Mode)
- 预设地图 + 固定波次 (10-25波)
- 每一关有独特主题和敌人配置
- 三星评定（通关/限时/无塔被摧毁）
- 通关解锁下一关和新内容
- 总计 20-30 关，分 4-5 个主题章节

#### 无尽模式 (Endless Mode)
- 所有已解锁内容可用
- 敌人无限波次，难度递增
- 每 5 波出现 Boss
- 排行榜（存活波数 / 击杀数 / 得分）
- 每日挑战（随机地图 + 随机限制条件）

### 3.3 游戏状态机

`GamePhase` 枚举控制行为切换：

```
Deployment → Battle → WaveBreak → (循环) → Victory/Defeat
```

- **Deployment**: 建造/升级/调整阵型，玩家手动触发"开始波次"
- **Battle**: 敌人行军，塔自动攻击（弹道系统），玩家操控移动单位
- **WaveBreak**: 当前波次结束，结算奖励，显示下一波预览
- **Victory/Defeat**: 关卡结束，评分结算

### 3.4 胜负条件

- **胜利**: 消灭关卡所有波次的敌人，基地未被摧毁
- **失败**: 基地血量归零（敌人走到终点 or 被远程敌人摧毁）
- **额外失败**（特定关卡）: 限时内未消灭所有敌人
