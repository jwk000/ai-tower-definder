# Tower Defender — 关卡设计文档

## 概述

游戏包含 5 个关卡，每个关卡拥有独特的场景主题、文字描述、障碍物布置和地格配色方案。关卡数据采用 JSON 风格的纯数据对象配置，便于维护和扩展。

## 设计原则

1. **数据驱动**: 所有关卡配置使用声明式 JSON 风格数据结构，不依赖运行时程序化生成
2. **场景叙事**: 每个关卡有详细的文字描述，说明场景地点、环境特征和面临的敌人
3. **视觉主题**: 地格配色方案严格匹配场景文字描述，增强沉浸感
4. **障碍物系统**: 场景中放置主题相关的装饰性障碍物，丰富视觉层次

## 数据结构

### MapConfig 扩展

```typescript
interface MapConfig {
  name: string;                    // 地图名称
  cols: number;                    // 列数 (21)
  rows: number;                    // 行数 (9)
  tileSize: number;                // 地格像素 (64)
  tiles: TileType[][];            // 地格类型二维数组
  enemyPath: GridPos[];           // 敌人路径点
  tileColors?: Partial<Record<TileType, string>>;  // 地格配色
  altSpawnPoints?: GridPos[];     // 备用出生点
  sceneDescription?: string;       // 场景文字描述
  obstaclePlacements?: ObstaclePlacement[];  // 障碍物布置
}
```

### ObstaclePlacement

```typescript
interface ObstaclePlacement {
  row: number;       // 所在行
  col: number;       // 所在列
  type: ObstacleType; // 障碍物类型
}
```

### ObstacleType 枚举

```typescript
enum ObstacleType {
  // 平原
  Tree = 'tree',
  Bush = 'bush', 
  Flower = 'flower',
  // 沙漠
  Rock = 'rock',
  Cactus = 'cactus',
  Bones = 'bones',
  // 冰原
  IceCrystal = 'ice_crystal',
  SnowTree = 'snow_tree',
  FrozenRock = 'frozen_rock',
  // 火山
  LavaVent = 'lava_vent',
  ScorchedTree = 'scorched_tree',
  VolcanicRock = 'volcanic_rock',
  // 城堡
  Pillar = 'pillar',
  Brazier = 'brazier',
  Rubble = 'rubble',
}
```

## 关卡设计

### 第1关 — 平原 · 村庄郊外

**场景描述**: 
青翠的草原上点缀着野花和灌木丛。一条土路从村庄延伸而来，蜿蜒穿过田野。这里是哥布林侦察兵经常出没的地方，村民们已经加固了道路尽头的基地。微风吹过，草浪起伏，看似平静的表面下暗藏杀机。

**敌人配置**:
- 小兵(Grunt): 哥布林步兵，数量众多
- 快兵(Runner): 哥布林侦察兵，移动迅速
- 重装兵(Heavy): 持盾哥布林，防御较高
- 法师(Mage): 哥布林萨满，远程攻击
- 自爆虫(Exploder): 被驱使的爆炸甲虫
- 指挥官(BossCommander): 哥布林小头目，第一波BOSS
- 攻城兽(BossBeast): 被驯化的巨兽，最终BOSS

**障碍物**: 树木、灌木丛、花丛 — 分布在空地格上
**配色**: 草地绿 `#3a7d44` / `#5a9e5f`, 土路棕 `#a1887f`

### 第2关 — 沙漠 · 商路遗迹

**场景描述**:
曾经繁荣的古代商路如今被黄沙半掩。巨大的砂岩和枯骨散落在道路两旁，仙人掌在烈日下顽强生长。沙漠强盗占据了这片区域，他们在岩石后方设伏，等待过往的商队。热浪扭曲了地平线，每一步都像是在火炉中行走。

**敌人配置**:
- 小兵(Grunt): 沙漠掠夺者
- 快兵(Runner): 沙地疾行者
- 重装兵(Heavy): 沙岩铠甲兵
- 法师(Mage): 沙漠术士
- 自爆虫(Exploder): 沙蝎
- 指挥官(BossCommander): 强盗头目
- 攻城兽(BossBeast): 沙虫巨兽

**障碍物**: 岩石(可建造格+阻挡格)、仙人掌、枯骨
**配色**: 沙地黄 `#c9a96e`, 深土路 `#8b7355`, 岩石褐 `#795548`

### 第3关 — 冰原 · 永冻之地

**场景描述**:
永恒的冬天笼罩着这片冰原。地面覆盖着厚厚的积雪，尖锐的冰晶从地表刺出，被冰封的古树如同幽灵般伫立。两条道路从不同的方向汇聚——敌人在此地建立了双线进攻的据点。寒风呼啸，连呼吸都会在空气中凝结成冰雾。

**敌人配置**:
- 小兵(Grunt): 冰原斥候
- 快兵(Runner): 冰面滑行者
- 重装兵(Heavy): 冰霜巨汉
- 法师(Mage): 寒冰术士
- 自爆虫(Exploder): 冰爆甲虫
- 指挥官(BossCommander): 冰霜将领 (出现两次)
- 攻城兽(BossBeast): 冰原猛犸

**障碍物**: 冰晶、雪松、冰封岩石
**配色**: 冰雪白 `#cfd8dc`, 冰蓝路 `#90a4ae`, 青色出生点 `#4dd0e1`

### 第4关 — 火山 · 炼狱深渊

**场景描述**:
活火山的斜坡上，岩浆河从裂缝中涌出，将黑曜石般的大地切割成支离破碎的岛屿。烧焦的树干像炭化的骨骼指向天空，火山砾石铺满了所有可以立足的地方。火山灰如黑雪般飘落，空气中弥漫着硫磺的气味。敌人在这种极端环境中如鱼得水。

**敌人配置**:
- 小兵(Grunt): 火元素步兵
- 快兵(Runner): 熔岩蜥蜴
- 重装兵(Heavy): 熔岩巨像
- 法师(Mage): 火焰术士
- 自爆虫(Exploder): 熔火甲虫
- 指挥官(BossCommander): 炎魔将领
- 攻城兽(BossBeast): 熔核巨兽

**障碍物**: 熔岩裂缝(阻挡格)、烧焦枯树、火山岩
**配色**: 焦土黑 `#4e342e`, 暗红路 `#8d6e63`, 岩浆红 `#d32f2f`

### 第5关 — 城堡 · 远古要塞

**场景描述**:
一座被遗忘的远古要塞内部。厚重的石墙和巨大的石柱构成了迷宫般的走廊，墙壁上的火炬是唯一的光源，投下摇曳的阴影。地面上散落着坍塌的碎石，敌人的主力部队已经占领了这座要塞，准备以此为据点发动最终进攻。这是最后的防线，退无可退。

**敌人配置**:
- 全部 5 种基础敌人类型
- 指挥官(BossCommander): 出现在第10波和第15波
- 攻城兽(BossBeast): 未出现(第5关以双指挥官+大量精英兵为主)

**障碍物**: 石柱(阻挡格)、火炬台(装饰)、碎石堆
**配色**: 暗石板 `#37474f`, 石板路 `#616161`, 纯黑墙 `#1a1a1a`, 金色出生点 `#fff176`

| 关卡 | 波数 | 初始金币 | 可用塔 | 特色 |
|------|------|----------|--------|------|
| 1-平原 | 10波 | 200 | 箭塔 | 引导关卡，逐步引入敌人类型 |
| 2-沙漠 | 12波 | 250 | 箭塔、炮塔 | 路径更长，加入阻挡格限制 |
| 3-冰原 | 15波 | 300 | 箭塔、冰塔 | 双路线，交替出兵点 |
| 4-火山 | 15波 | 350 | 箭塔、炮塔、电塔 | 密集岩浆阻挡，复杂路径 |
| 5-城堡 | 15波 | 400 | 全部四塔 | 迷宫路径，精英混编，双BOSS |

## 渲染实现

障碍物在 `RenderSystem.drawMap()` 中绘制：
- 每个障碍物根据其类型使用不同的形状和颜色
- 障碍物绘制在地格之上、实体之下
- 障碍物仅用于装饰，不影响游戏逻辑（阻挡通过 `TileType.Blocked` 实现）

地格配色通过 `MapConfig.tileColors` 实现逐关定制，替代了原来的硬编码颜色。
